# Day 01 - Basics

## 🔹 Ansible Task — Install Nginx on Pathnex Server

Rewrite this YAML manually:

---
- name: Install Nginx on Pathnex server
  hosts: all
  become: yes

  tasks:
    - name: Install nginx
      yum:
        name: nginx
        state: present

# Git Integration
## Jenkins Pipeline - Checkout Git
You will learn how to **Checout a Git repository** and list files.


pipeline {
   agent any
   stages {
	stage {
		git branch: 'main', url: 'https://github.com/Pathnex/sample-java-app.git'
	}
   }
    stage('List Files') {
        steps {
            sh 'la -la'
         }
     }
   }
}
